# my-framework



## 快速入门

```bash
$ npm install
$ npm test
```

在应用中声明使用框架：

```js
// {app_root}/index.js
require('my-framework').startCluster({
  baseDir: __dirname,
  // port: 7001, // default to 7001
});

```
