# cnode-api

cnode RESTful API

## QuickStart

### Development
```shell
$ npm install
$ npm run dev
$ open http://localhost:7001/
```

### Deploy

Use `EGG_SERVER_ENV=prod` to enable prod mode

```shell
$ EGG_SERVER_ENV=prod npm start
```

### npm scripts

- Use `npm run autod` to auto detect dependencies upgrade
- Use `npm run lint` to
- Use `npm test` to run unit test
