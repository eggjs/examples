# hackernews-datahub

---

## Thanks To

- use [Macaca](//macacajs.github.io) for UI E2E testing.
- use [DataHub](//github.com/macacajs/macaca-datahub) for mock tool and proxy service.

## Development

```bash
$ npm install
$ npm run dev
$ open http://localhost:7001/
```

## Unit Test

```bash
$ npm run test
```

## E2E Test

```bash
$ npm run datahub
$ npm run dev:e2e
$ npm run test:e2e
```
